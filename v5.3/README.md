# CPC464-2MINI

CPC464-2MINI v5.3 Main Board

# Main points for this release

* AMSGateArray Xilinx based Gate Array
* 128K RAM
* SMD 40031 GAL required for 128K operation
* QFP44 Z80 CPU
* AY-3-8910 Sound/IO Only
* PLCC32 ROM with switch selectable banking
